Panduan Instalasi Fedora Admin

Langkah-langkah memasang Fedora Linux untuk sistem server parkir otomatis

1. Persiapan Awal

  • Download Fedora Workstation atau Server di situs resmi: getfedora.org
  • Buat bootable USB dengan Rufus atau BalenaEtcher
  • Siapkan komputer server (minimal RAM 4GB, SSD 64GB, LAN port)

2. Langkah Instalasi

  1. Masukkan USB dan boot ke installer Fedora
  2. Pilih bahasa dan klik "Install to Hard Drive"
  3. Atur partisi secara otomatis (default) atau manual jika dual boot
  4. Set lokasi, keyboard, dan timezone
  5. Buat user admin (misalnya: user `cmadmin`, password `admin123`)
  6. Set hostname server, misalnya: `server-parkir.local`
  7. Klik “Begin Installation” dan tunggu hingga selesai
  8. Reboot dan cabut USB

3. Setting Setelah Instalasi

  • Lakukan update sistem: sudo dnf update
  • Install software penting: sudo dnf install net-tools vim openssh-server
  • Aktifkan SSH agar bisa remote: sudo systemctl enable --now sshd
  • Cek IP address: ip addr
  • Tambahkan firewall rule jika perlu

4. Integrasi ke Sistem Parkir

Fedora digunakan sebagai server utama untuk:

  • Menyimpan database transaksi parkir
  • Hosting sistem backend (API, dashboard monitoring)
  • Menjadi server kamera CCTV (via NVR software)
  • Komunikasi dengan client POS melalui jaringan lokal
