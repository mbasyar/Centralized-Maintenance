Konfigurasi Barrier Gate (Palang Otomatis)

Panduan pemasangan dan integrasi barrier dengan sistem kontrol parkir

1. Komponen Terkait

  • Barrier Gate (contoh: MAG BR500, CAME, Taliwang)
  • Modul relay controller (USB/Serial)
  • POS Client (dengan software kontrol gate)
  • Kabel data RS232/RS485 atau TCP/IP
  • Catu daya / power supply

2. Wiring dan Pemasangan

  1. Pasang barrier di lokasi gate masuk/keluar
  2. Sambungkan kabel “Open” dan “Close” ke relay
  3. Hubungkan modul relay ke komputer POS (via USB/COM)
  4. Pastikan ground listrik tersambung ke barrier & kontroler
  5. Aktifkan sensor safety (loop detector / infrared sensor)

3. Pengaturan di Software

  • Buka POS Client → masuk ke menu “Device Settings”
  • Pilih port relay (COM3/COM4 tergantung PC)
  • Atur waktu buka/tutup otomatis (misalnya: buka 5 detik)
  • Uji coba tombol manual OPEN dan CLOSE dari software
  • Cek respons barrier saat kartu/tiket berhasil diproses

4. Troubleshooting Umum

  • Jika barrier tidak merespons, cek power supply & sambungan kabel
  • Cek port COM di Device Manager (Windows)
  • Pastikan tidak ada konflik COM port antar perangkat
  • Gunakan multimeter untuk memastikan output relay bekerja
  • Pastikan sensor safety tidak menghalangi proses penutupan

5. Tips Instalasi

  • Gunakan box kontrol yang aman dari cuaca dan gangguan fisik
  • Selalu uji sistem sebelum diaktifkan secara live
  • Siapkan tombol manual override untuk keadaan darurat
