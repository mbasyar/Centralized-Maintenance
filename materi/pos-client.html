Instalasi & Konfigurasi POS Client

Digunakan di komputer gate parkir untuk mengelola tiket & akses kendaraan

1. Persiapan

  • Komputer client dengan OS Windows (7/10)
  • Koneksi LAN ke server Fedora
  • Installer POS Client (*.exe), biasanya dari vendor
  • Printer thermal (untuk cetak tiket)
  • Reader kartu/member (opsional)

2. Instalasi Software

  1. Jalankan installer POS Client
  2. Pilih direktori instalasi (biarkan default jika tidak yakin)
  3. Ikuti wizard sampai selesai
  4. Restart komputer setelah instalasi

3. Setting Koneksi ke Server

  • Buka aplikasi POS Client
  • Masuk ke menu “Settings” atau “Configuration”
  • Isi IP Server: `192.168.1.10` (contoh IP Fedora Server)
  • Pilih Port komunikasi: misalnya `8080` atau `3306` untuk MySQL
  • Test koneksi: pastikan status "Connected"

4. Integrasi Perangkat

  • Pasang printer thermal ke port USB
  • Install driver printer jika belum otomatis terdeteksi
  • Uji cetak tiket dari menu POS
  • Pasang card reader (jika pakai RFID/Mifare), dan sambungkan via USB
  • Uji pembacaan kartu

5. Tips Tambahan

  • Gunakan aplikasi VNC untuk remote konfigurasi POS dari server
  • Hindari restart mendadak saat proses transaksi berlangsung
  • Backup konfigurasi POS secara berkala
